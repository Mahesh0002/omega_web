# Project OMEGA

**The Sovereign Discovery Engine**

A White-Box, Neuro-Symbolic Artificial Intelligence architecture designed for Symbolic Regression. Project OMEGA reverse-engineers the exact, interpretable mathematical laws governing chaotic systems (like the Double Pendulum) from raw observational telemetry data.

## Key Concepts

- **The Trinity Architecture**: An asynchronous, co-evolutionary loop featuring:
  - **The Explorer**: A Transformer-based Actor-Critic network with Monte Carlo Tree Search (MCTS) for structural syntax.
  - **The Alchemist**: An L-BFGS optimizer injecting constants into equations and fine-tuning them.
  - **The Shadow**: A Meta-Controller mitigating engine stagnation via Chaos Events.
- **The Iron Dome**: Rigorous dimensional analysis and pruning to halt MCTS combinatorial explosions.
- **Protocol Zero**: An elastic co-evolution algorithm stacking recursive search trees on continuous force residuals to solve deep chaos.

## Tech Stack

- **Framework**: React 19 + TanStack Start (SSR)
- **Routing**: TanStack Router
- **Styling**: Tailwind CSS v4 + custom glowing typography & glassmorphism
- **Animations**: GSAP, Three.js
- **Build Tool**: Vite

## Getting Started

1. **Install dependencies**
   ```bash
   npm install
   # or with bun
   bun install
   ```

2. **Run the development server**
   ```bash
   npm run dev
   # or
   bun run dev
   ```

3. **Build for production**
   ```bash
   npm run build
   # or
   bun run build
   ```

## Project Structure

- `src/routes/` - TanStack Router definitions for Home, Architecture, and Engineering logs.
- `src/components/` - Advanced simulation components:
  - `OmegaAscension.tsx` - High-fidelity live MCTS reasoning simulation.
  - `OmegaDiscovery.tsx` - GSAP-based SVG discovery engine sequence.
- `src/styles.css` - Global theme definitions, styling tokens, and cyan/neon aesthetic overlays.
